# 🚀 Abhishek Singha's Portfolio Website

I have successfully created a modern, premium portfolio website for **Abhishek Singha** using **React**, **Material UI**, and **Vite**. The design is fully responsive (Material UI), mobile-friendly, and optimized for performance.

## ✨ Features
- **Modern Material UI**: Clean, professional aesthetic with a deep-blue and white theme.
- **Responsive Design**: Looks great on desktops, tablets, and smartphones.
- **Sectioned Layout**:
  - **Hero**: High-impact introduction with a professional profile portrait.
  - **Experience**: Detailed timeline of 9+ years in academic and faculty roles.
  - **Education**: Showcasing degrees from Bundelkhand University, Stanford, and AIU Ireland.
  - **Certifications**: A dedicated grid for the 30+ specialized certificates.
  - **Awards**: Highlighting the Visionary, International Icon, and Ratna awards.
  - **Contact**: Professional contact portal with info and a message form.
- **Smooth Navigation**: One-page scrolling with a sticky blurred navbar.

## 🛠 Tech Stack
- **React 19**
- **Material UI v7** (latest)
- **Vite** for fast builds
- **gh-pages** for one-command deployment

## 🌐 How to Publish to GitHub Pages
Follow these simple steps:

1. **Initialize Git & Link Repo**:
   ```bash
   git init
   git add .
   git commit -m "Initialize portfolio"
   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
   ```

2. **Deploy to GitHub Pages**:
   Run:
   ```bash
   npm run deploy
   ```
   *This will automatically build the site and push it to the `gh-pages` branch.*

3. **Enable Pages on GitHub**:
   - Go to your Repo **Settings > Pages**.
   - Ensure the source is set to the `gh-pages` branch.
